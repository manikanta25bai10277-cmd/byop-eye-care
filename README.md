# AI-Based Eye Care Reminder System

## 📌 Description

This project is an AI-based Eye Care Reminder System designed to help users reduce eye strain caused by prolonged screen usage. It provides intelligent reminders based on user screen time habits.

## 🎯 Problem Statement

Many students and professionals suffer from eye strain due to continuous screen exposure. There is a need for a simple system that reminds users to take breaks and care for their eyes.

## 💡 Solution

This system uses a rule-based approach to analyze screen usage and automatically set reminder intervals for eye care activities.

## ⚙️ Features

* User input for daily screen time
* Intelligent reminder interval selection
* Continuous eye care alerts
* Data logging using file handling

## 🧠 AI Concept Used

Rule-based decision system:

* If screen usage > 6 hours → frequent reminders
* Else → moderate reminders
